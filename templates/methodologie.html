Méthodologie de Travail

Approche de développement

  • Méthode itérative et collaborative inspirée des méthodes agiles
  • Binôme permanent avec validation des étapes
  • Wireframes & MLD validés avant développement
  • Intégration continue avec tests locaux

Environnement technique

  • Backend : Flask (Python), SQLAlchemy
  • Base de données : MySQL
  • Frontend : Flask Templates, HTML/CSS/JS natif
  • Outils : Git/GitHub, Balsamiq, Looping, Postman
  • Sécurité : JWT, RBAC, CSRF protection

Workflow Git

  • Branches : main (stable), dev (intégration), feature/... (tâches)
  • Pull requests systématiques avec relecture de code
  • Tests avant chaque merge
  • Exclusion des fichiers sensibles (config.py)
